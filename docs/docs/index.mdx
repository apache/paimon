## Why Paimon?

<div className="feature-grid">
<div className="feature-item">

### Large-Scale Analytics
Petabyte-scale tables with time travel, fast scan planning, schema evolution, and incremental clustering.

</div>
<div className="feature-item">

### Realtime Streaming
LSM-powered streaming updates with multiple merge engines (Deduplicate, Partial Update, Aggregation) and changelog producers.

</div>
<div className="feature-item">

### Multimodal AI
Store images, videos, embeddings in blob tables. Vector search (DiskANN), full-text search, and native Python SDK for ML pipelines.

</div>
</div>
